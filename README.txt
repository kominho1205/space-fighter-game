# 온라인 전투기 대전 웹 게임

키보드 방향키와 스페이스바로 조작하는 1vs1 온라인 전투기 게임입니다.  
랜덤 매칭으로 동시에 접속한 상대와 자동으로 매칭됩니다.

## 기능 요약

- 방향키 ←↑↓→ : 전투기 상하좌우 이동
- 스페이스바 : 탄약을 소모하여 총 발사
- 탄약: 게이지 형식, 발사 시 소모되고 시간에 따라 자동 충전
- 목숨: 하트 3개 (피격 시 1개 감소, 0이 되면 패배)
- 아이템
  - 하트 아이템: 목숨 +1 (최대 3개)
  - 방어막 아이템: 2초간 무적
- 다시 시작 버튼: 게임 종료 후 양쪽 모두가 다시 시작을 요청하면 같은 상대와 재대전

## 실행 방법

1. Node.js 설치
   - Node.js가 설치되어 있지 않다면, Node.js 공식 사이트에서 LTS 버전을 설치합니다.

2. 프로젝트 폴더 준비
   - 이 폴더 전체를 원하는 위치에 저장합니다.
   - 폴더 구조 예시:
     - space-fighter-game/
       - package.json
       - server.js
       - public/
         - index.html
         - style.css
         - client.js

3. 의존성 설치
   - 터미널(또는 CMD, PowerShell)을 열고 프로젝트 폴더로 이동합니다.
     - 예: `cd space-fighter-game`
   - 다음 명령어 실행:
     - `npm install`

4. 서버 실행
   - 다음 명령어 실행:
     - `npm start`
   - 터미널에 `Server running on port 3000` 이 표시되면 준비 완료입니다.

5. 브라우저에서 접속
   - 크롬 등 원하는 브라우저에서 아래 주소로 접속합니다.
     - `http://localhost:3000`
   - 브라우저 창/탭을 두 개 열고 같은 주소로 접속하면, 두 클라이언트가 서로 매칭되어 1vs1 대전을 테스트할 수 있습니다.

6. 배포 (선택)
   - Render, Railway, Heroku, VPS 등 Node.js를 실행할 수 있는 환경에 이 프로젝트를 올리면 인터넷 상에서 다른 사람들과도 플레이할 수 있습니다.
   - 배포 시에는 `npm install` 후 `npm start` 또는 서버 제공업체의 Node.js 런타임 설정에 맞게 설정하면 됩니다.
